# Arena Torcidas Portal Completo

Projeto Next.js já preparado com:

- Portal estilo esportivo
- Classificação automática
- Partidas e rodadas
- Mata-mata
- Clubes com escudos
- Jogadores e estatísticas
- Notícias
- Regulamento
- Streamers
- Painel administrativo
- Backup local

## Instalação no Codespaces

1. Apague o conteúdo atual do repositório, mantendo apenas a pasta `.git`.
2. Envie todos os arquivos desta pasta para a raiz do repositório.
3. No terminal, execute:

```bash
npm install
npm run dev
```

4. Abra a porta 3000 quando o Codespaces oferecer.
